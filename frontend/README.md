#  Canopy

Canopy is a forest tracking web application designed to monitor, organise, and visualise forest-related data in a clear and user-friendly way. The app focuses on clean frontend architecture, thoughtful state management, and scalable UI patterns.

>  **Status:** In Progress  
> Core features are functional, with additional capabilities and refinements planned.

---

##  Features

- Forest and location data tracking
- Dynamic data fetching with loading and error states
- Responsive, map-friendly layouts
- Reusable, component-driven UI
- Clean separation of UI, logic, and data layers

---

##  Purpose

Canopy was built to:
- Practice building **data-driven React applications**
- Work with structured environmental data
- Apply scalable frontend architecture patterns
- Focus on usability and clarity for information-heavy interfaces

The project prioritises maintainability and clarity over rushed feature completion.

---

##  Tech Stack

- **Frontend:** React
- **Styling:** Tailwind CSS / shadcn/ui
- **State Management:** React Context / Local State
- **Data Fetching:** Fetch API / Axios
- **Version Control:** Git & GitHub
